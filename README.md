

# **Web Scraping with Python & BeautifulSoup**

# Amazon Product Data Scraper*

This project is a beginner-friendly introduction to web scraping using Python and BeautifulSoup. It demonstrates how to extract real-time product information from Amazon and organize it into a structured dataset. The goal of this project is to help learners understand how web scraping works in real-world scenarios.

---

## **Project Overview**

The project focuses on scraping essential product details from Amazon, such as title, price, ratings, reviews, and availability. It is designed to help beginners understand how websites structure their content and how data can be extracted and used for analysis.

---

## **Key Features**

* Scrapes product information from Amazon
* Extracts product title, price, ratings, reviews, and stock status
* Generates clean and structured output
* Beginner-friendly explanation and workflow
* Useful for learning data extraction and analysis fundamentals

---

## **Technologies Used**

* Python
* Requests Library
* BeautifulSoup (bs4)
* CSV Format (for storing results)
